Architectural Partitioning: How We Slice the Cake

Technical Partitioning

Code is grouped by its technical role. This is the classic Layered Architecture.

A business feature like 'Checkout' is spread across all layers.

Domain Partitioning

Code is grouped by its business capability. This is core to DDD, Modular Monoliths, and Microservices.

All code for 'Checkout' lives in one module.
